# Proyecto Java Hola Mundo

Este proyecto contiene un ejemplo simple en Java que imprime un mensaje de bienvenida y muestra la suma de los números del 1 al 100.

## Requisitos

- Java Development Kit (JDK) instalado
- Consola o terminal para compilar y ejecutar el programa

## Ejecución

1. Compila el archivo:
   ```bash
   javac HolaMundo.java
   ```
2. Ejecuta el programa:
   ```bash
   java HolaMundo
   ```

## Archivos del proyecto

- `HolaMundo.java`: archivo principal con el programa Java.
- `README.md`: documentación del proyecto.
